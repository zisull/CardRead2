# Cardread Pro

> 一款本地电子书阅读器，支持 TXT / Markdown / EPUB / MOBI 格式，拥有 6 种主页布局、20 种内置主题、自动阅读、全文检索、便签系统等丰富功能。

## 下载安装

### 直接下载（推荐）

从 [GitHub Releases](https://github.com/zisull/CardRead2/releases) 或 [蓝奏云](https://zisull.lanzoub.com/b02xfay1ah?pwd=cz4a) 下载对应平台的可执行文件：

| 平台 | 文件 | 说明 |
|------|------|------|
| Windows x64 | `CardRead2-Windows-x64.exe` | 单文件，双击即用 |
| Linux x64 | `CardRead2-Linux-x64` | 需安装 `webkit2gtk` |
| macOS Intel | `CardRead2-macOS-x64` | Intel 芯片 Mac |
| macOS Apple Silicon | `CardRead2-macOS-arm64` | M1/M2/M3/M4 芯片 Mac |

### 从源码运行

```bash
git clone https://github.com/zisull/CardRead2.git
cd CardRead2
pip install -r requirements.txt
python src/web/main.py
```

## 支持格式

| 格式 | 扩展名 | 特性 |
|------|--------|------|
| **TXT** | `.txt` `.text` `.log` | 纯文本，自动识别编码（GBK/UTF-8/GB2312 等） |
| **Markdown** | `.md` `.markdown` | 完整 Markdown 渲染，支持图片、代码高亮 |
| **EPUB** | `.epub` | 电子书标准格式，支持内链封面、图片灯箱查看 |
| **MOBI** | `.mobi` `.pdb` | Kindle 格式，基于 KindleUnpack 自动解析 |

## 主界面

主界面提供 **6 种布局风格**，可在设置中自由切换。每种布局都包含相同的核心功能区域。

### 布局一览

| 布局 | 风格 | 特点 |
|------|------|------|
| **默认** | 经典列表 | Hero 继续阅读卡 + 标准列表，简洁实用 |
| **极简风** | 优雅编号 | 大量留白，列表风格，专注阅读本身 |
| **仪表盘** | HUD 控制台 | 六边形图标 + 环形图表 + 冷蓝配色，科技感十足 |
| **国风韵** | 水墨意境 | 印章装饰 + 楷体排版 + 文房意境，中式美学 |
| **航海志** | 大航海 | 船舵/锚元素 + 深蓝配色 + 罗盘装饰 |
| **黑胶片** | 复古唱片 | 旋转唱片动画 + 黑红经典配色，音乐质感 |

### 顶部导航栏

书架页面顶部导航栏提供以下入口：

| 图标 | 功能 | 说明 |
|------|------|------|
| 📚 | **书架** | 默认视图，管理所有书籍 |
| 🔖 | **书签** | 查看所有书籍的书签 |
| 📝 | **便签** | 打开便签管理面板 |
| 📊 | **统计** | 阅读数据统计 |
| ⚙ | **设置** | 主题/字体/排版设置 |
| ❓ | **说明** | 功能说明与使用指南 |
| 📋 | **日志** | 应用运行日志 |

### 右侧统计面板

- 总藏书数
- 总章节数
- 累计阅读时长
- 平均阅读进度

### 搜索与筛选

- **搜索框**：支持书名搜索，输入拼音首字母（如 `SDYXZ`）或全拼音（`shediaoyingxiongzhuan`）可匹配中文书名
- **格式筛选**：按 TXT / MD / EPUB / MOBI 过滤
- **进度筛选**：未读 / 阅读中 / 已读
- **排序方式**：最近阅读 / 书名字母序 / 添加时间 / 阅读进度

### 导入书籍

两种方式导入书籍：

1. **点击「+」按钮**：打开文件对话框，可多选文件
2. **拖放文件**：直接将文件拖放到窗口，会显示拖放提示覆盖层

支持批量导入，同名书籍自动跳过，单文件上限 200MB。

### 右键菜单

在书籍卡片上右键弹出菜单：

| 选项 | 功能 |
|------|------|
| 打开阅读 | 进入阅读器 |
| 打开目录 | 在文件管理器中定位文件 |
| 复制书名 | 复制到剪贴板 |
| 复制文件路径 | 复制完整路径 |
| 重命名 | 修改显示名称 |
| 查看详情 | 文件名/大小/格式/章节/字数/进度 |
| 删除书籍 | 删除文件及所有关联数据 |

## 阅读界面

点击任意书籍进入阅读界面，提供沉浸式阅读体验。

### 窗口控件

阅读界面顶部工具栏：

| 控件 | 功能 |
|------|------|
| 书名标题 | 显示当前阅读的书籍名称，可拖拽移动窗口 |
| 🌙/☀ | 切换深色/浅色主题 |
| 🔍 | 打开搜索面板（Ctrl+F） |
| 🔖 | 打开书签面板 |
| 📌 | 置顶/取消置顶窗口 |
| — | 最小化 |
| □ | 最大化/还原 |
| ✕ | 关闭 |
| ↩ | 返回上次位置（出现时显示，带关闭按钮） |

### 翻页操作

| 操作 | 效果 |
|------|------|
| 点击页面左侧 / `←` / `PageUp` | 上一页 |
| 点击页面右侧 / `→` / `PageDown` / `空格` | 下一页 |
| `↑` / `↓` 方向键 | 逐行滚动 |
| 滚轮 | 正常滚动 |
| 章节首尾继续翻页 | 自动切换到上/下章 |

### 阅读进度

- 进度自动保存，下次打开自动恢复
- 底部状态栏显示当前章节名、进度百分比
- 打开书籍时如检测到上次位置，右下角显示「↩ 返回上次位置」浮层，点击跳转或点 × 关闭

### 字体与排版

在设置面板中可调整：

| 项目 | 范围 | 说明 |
|------|------|------|
| 字体族 | 系统字体 + 自定义 | 6 种内置字体 + 自定义字体 |
| 字号 | 10 – 40 px | 也支持 Ctrl+Alt+滚轮 快捷调节 |
| 行距 | 1.0 – 3.0 | 行间距倍数 |
| 段距 | 0 – 40 px | 段落间距 |
| 首行缩进 | 0 – 4 em | 中文排版常用 2em |
| 文字颜色 | 自定义 | 阅读区域文字颜色 |

**自定义字体**：点击字体下拉框旁的 `＋` 按钮，选择 `.ttf` / `.otf` / `.woff` / `.woff2` / `.ttc` 字体文件，自动复制到字体文件夹并即时生效。下拉列表中自定义字体右侧带 `✕` 按钮，点击可移除并删除。点击 `📂` 可直接打开字体文件夹。

### 背景图片

- 可为主页和阅读界面分别设置背景图片
- 支持调整背景图片透明度（0 – 1）
- 支持调整背景图片缩放模式（cover / contain / auto）

## 自动阅读

自动阅读功能让页面匀速滚动，解放双手。

### 启动方式

- **右键菜单** → 选择「自动阅读」
- 阅读时使用快捷键启动

### 速度控制

| 操作 | 效果 |
|------|------|
| `Ctrl` + `↓` | 加速向下（+0.25） |
| `Ctrl` + `↑` | 减速 / 反向向上（-0.25） |
| `Ctrl` + 滚轮 | 同上，调节速度 |
| `Esc` | 停止自动阅读 |

- 速度范围：-20 到 20（正数向下滚动，负数向上滚动）
- 浮动徽章实时显示当前速度
- 到达章节末尾自动切换下一章

## 搜索功能

### 当前章节搜索

- 快捷键 `Ctrl+F` 打开搜索面板
- 在当前加载的文本中查找匹配
- 高亮显示所有匹配项
- `↑` / `↓` 按钮在匹配项间跳转
- `Esc` 关闭搜索

### 全文检索

- 搜索面板中点击「全文检索」按钮
- 基于 SQLite FTS5 全文索引，搜索整本书所有章节
- 结果列表显示章节名 + 匹配上下文
- 点击结果项跳转到对应位置并高亮
- 支持拼音搜索

## 便签系统

便签是独立于书籍的笔记功能，支持 Markdown 编辑和桌面展示。

### 便签管理面板

| 功能 | 说明 |
|------|------|
| 新建便签 | 创建空白便签 |
| 搜索 | 全文搜索便签内容 |
| 分组过滤 | 按分组筛选显示 |
| 排序 | 按更新时间 / 创建时间 / 标题 |
| 批量操作 | 全选、批量删除、清空 |
| 复制 | 复制便签内容 |
| 导出 | 导出为 `.md` 文件 |
| 重命名 | 双击标题编辑 |
| 分组编辑 | 双击分组标签修改 |

### 便签编辑器

独立窗口（720×520），提供 Markdown 编辑体验：

- **左右分栏**：左侧编辑、右侧实时预览，分隔线可拖拽调整比例
- **工具栏**：粗体 / 斜体 / H1 / H2 / 无序列表 / 有序列表 / 待办清单 / 代码块 / 链接
- **快捷键**：`Ctrl+S` 保存、`Ctrl+B` 粗体、`Ctrl+I` 斜体、`Tab` 缩进
- **自动保存**：编辑停止 1 秒后自动保存
- **字数统计**：底部实时显示
- **代码高亮**：预览区代码块自动语法高亮

### 桌面展示

独立置顶悬浮窗口（320×420）：

- Markdown 渲染展示
- 可切换为编辑模式
- 适合做桌面便签提醒

## 技术架构

```
┌─────────────────────────────────────────────┐
│                  pywebview                   │
│         (EdgeChromium / WebKitGTK / Cocoa)   │
├─────────────────────────────────────────────┤
│  HTML / CSS / JS   ←→   Python API (bottle) │
├─────────────────────────────────────────────┤
│  Parsers    │  Reading Engine  │  Themes     │
│  TXT/MD/    │  Chapter Parse   │  20 Built-in│
│  EPUB/MOBI  │  FTS5 Search     │  Custom     │
├─────────────────────────────────────────────┤
│  SQLite (confull)  │  Loguru  │  chardet    │
└─────────────────────────────────────────────┘
```

- **前端**：原生 HTML / CSS / JavaScript，无框架依赖
- **后端**：Python 3.10+，pywebview 桌面窗口
- **数据库**：SQLite，FTS5 全文索引
- **构建**：Nuitka 编译为单文件可执行程序
- **跨平台**：Windows / macOS / Linux
